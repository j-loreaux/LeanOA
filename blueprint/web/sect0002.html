0.2 Order Lemmas

Let \(\mathcal{A}\) be a unital \(C^{*}\)–algebra in this section. We collect lemmas for the ordering of elements in \(\mathcal{A}\). Recall that a selfadjoint element \(a \in \mathcal{A}\) is said to be positive if its spectrum is contained in \(\mathbb {R}_{\ge 0}\). This is written \(a\ge 0\). If \(a,b \in \mathcal{A}\) are selfadjoint then \(b \le a\) if \(b-a \ge 0\). (TO DO: adapt this to Jireh’s fully general ordering on all elements without mentioning the selfadjoint ones.)
