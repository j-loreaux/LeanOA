0.3 Projection Lemmas

Let \(\mathcal{A}\) be a unital \(C^{*}\)–algebra in this section. In this section we collect relevant results about (selfadjoint) projections in \(\mathcal{A}\). Recall that an element \(p\) in a \(C^{*}\)–algebra is a projection if \(p^2=p^{*}=p\). By the Spectral Mapping Theorem (or using the full Gelfand Duality), a selfadjoint element \(a\in A\) is a projection if and only if the spectrum of \(a\) is contained in \(\{ 0,1\} \).

Lemma 5

Uses

  • Lemma 4

Let A be a \(C^{*}\)–algebra and \(p,q \in \mathcal{A}\) be projections. Then \(p-q\ge 0\) iff \(qp = pq = q\).

Proof ▶

By Lemma 2, if \(p-q \ge 0\) then \(q(p-q)q=qpq-q^3=qpq-q \ge 0\). By Lemma 4, we have \(p\le 1\) and employing Lemma 2 we obtain \(qpq\le q1q=q\). Since \(qpq\ge q\) and \(qpq\le q\) we have \(qpq=q\), which implies that \(q(p-q)q=0\). By the \(C^{*}\)-property of the norm, we have \(\| (p-q)^{1/2}q\| ^2=\| q(p-q)q\| =0\) and so \((p-q)^{1/2}q=0\) and therefore \((p-q)q=(p-q)^{1/2}(p-q)^{1/2}q=0\). It follows that \(pq=q\), and taking adjoints, \(qp=q\). Conversely, if \(qp=pq=q\), one easily checks that \((p-q)\) is a projection and so its spectrum is contained in \(\{ 0,1\} \) and it is positive.
